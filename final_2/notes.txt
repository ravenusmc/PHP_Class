Project Description

Create database tables to store all information for this application; user login, comments, replies, etc.

Insert the following user logins:
userid = user1, password = pwd1
userid = user2, password = pwd2
Create the necessary pages, forms, input elements, links, styles, etc. to allow users to login, and to view, add and delete comments and replies.
Validate the login against the user login table.

Display a login button or link if a user has not logged in.
Display a logout button or link if a user has logged in.
Display the userid of the user that has logged in.

Once a user has logged in, allow them to add and delete comments and replies until they logout or close their browser.
Any visitor can view all comments and replies. Login is not required to view comments and replies.
Visitors that do not login can only view comments and replies; and cannot add or delete comments or replies.

Login is required to add or delete comments and replies.
User login can modify or delete any comment or reply.
Display comments and replies.
Sort comments by newest first.
Sort replies by oldest first.
Display replies under their associated comment.
Display the userid of the user that entered the comment or reply.
Display the date and time that the user entered the comment or reply.
Use CSS styles to format your pages; fonts, colors, etc.

BONUS: 
Display replies indented under their associated comment.
Display the total number of comments.
Display the total number of replies for each comment.
